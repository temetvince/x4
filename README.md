## Exe Flags
* Run game in background:
    * -nosoundthrottle -nocputhrottle
* Skip intro:
    * -skipintro
* Debug All:
    * -debug all -logfile debuglog.txt
* Debug Scripts:
    * -debug scripts -logfile debuglog.txt
